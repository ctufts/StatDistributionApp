Beta Distribution

Overview

Used to describe a prior belief with two outcomes. P(theta|a, b) where theta is Two Parameters:

  • beta: number of outcome #1
  • alpha: number of outcome #2
  • Both are shape parameters

  • batting average : http://varianceexplained.org/statistics/beta_distribution_and_baseball/

Interesting Properties
